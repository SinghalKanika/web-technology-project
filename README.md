# 🎓 E-Learning Platform (Web Technology Project)

A simple **role-based E-Learning web application** built using **HTML, CSS, and JavaScript**, focusing on **Access Control** and **Content Delivery**.

## 📌 Features

### 🔐 Access Control (Primary Focus)

* User Authentication (Login / Signup)
* Role-Based Access:

  * 👨‍🎓 Student → View courses
  * 🧑‍🏫 Admin → Add/Delete courses
* Protected routes (Admin panel restricted)

---

### 📦 Content Delivery (Secondary Focus)

* Course listing system
* Course content viewer
* Dynamic content rendering
* Local storage-based persistence

---

## 🧩 Project Structure

```
/project
│── index.html        # Login / Signup page
│── dashboard.html    # User dashboard
│── courses.html      # Course listing
│── player.html       # Course content viewer
│── admin.html        # Admin panel
│── style.css         # Styling
│── script.js         # Logic & functionality
```
